### VeLiveQuickStartDemo 介绍

    VeLiveQuickStartDemo 基于 TTSDK 直播 SDK 开发，目前接入了基础、高级、互动三大功能模块。提供示例参考代码，后续会持续迭代。
    
    基础功能模块包含：基础推拉流能力
    
    高级功能模块包含：美颜推流、RTM 超低延迟推流、RTM 超低延迟拉流、自采集推流、H265 硬编码推流、码率自适应推流、屏幕采集推流、画中画
    
    互动功能模块包含：主播和观众连麦、主播和主播 PK
